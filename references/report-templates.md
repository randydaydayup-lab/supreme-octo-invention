## 质量检查清单

### 任务启动检查清单
- [ ] 监测目标明确且可衡量
- [ ] 样本量要求已确定（常规/专项）
- [ ] 交付时间合理（至少留出3个工作日）
- [ ] 任务单已审批

### 数据采集检查清单
- [ ] 样本数量达标（常规≥50条，专项≥200条）
- [ ] 平台分布均衡（≥3个平台，单平台<60%）
- [ ] 时效性符合要求（90%在30天内）
- [ ] 已过滤营销内容和机器人评论
- [ ] JSON格式验证通过

### 心智分析检查清单
- [ ] 有效样本率 ≥ 70%
- [ ] 所有9个维度均有数据填充
- [ ] 情感强度分布合理（无全极端值现象）
- [ ] 心智标签无重复或乱码
- [ ] 过滤率 < 30%（否则需人工复核）

### 报告交付检查清单
- [ ] 核心洞察 ≥ 3个，每个洞察样本量≥10条
- [ ] 每个洞察都有具体数字支撑
- [ ] 至少引用5条原始用户评论（已脱敏）
- [ ] 产品建议可落地（含短/中/长期）
- [ ] 图表清晰可读，标注完整
- [ ] 用户隐私信息已脱敏处理
- [ ] 负责人审批通过

## KPI监控指标

| 指标 | 目标值 | 预警阈值 | 监控频率 |
|-----|-------|---------|---------|
| 任务按时交付率 | ≥ 90% | < 80% | 每周 |
| 有效样本率 | ≥ 70% | < 50% | 每次任务 |
| 洞察报告采纳率 | ≥ 60% | < 40% | 每月 |
| 数据采集效率 | ≤ 4小时/50条 | > 6小时 | 每次任务 |
